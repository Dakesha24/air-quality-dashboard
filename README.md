## Deskripsi
Dashboard ini dibuat untuk menganalisis kualitas udara di berbagai stasiun di Beijing selama periode 2013-2017. Projek ini mencakup analisis tren PM2.5 dan korelasi polutan dengan cuaca.

## Struktur Direktori
- `/dashboard`: Berisi data hasil cleaning dan file python untuk dashboard.
- `/data`: Berisi dataset mentah (CSV per stasiun).
- `notebook.ipynb`: Berkas analisis data lengkap.
